Class: NlpManager

NlpManager(settings)

Class for the NLP Manager. The NLP manager is the one that is able to manage several classifiers, to have multilanguage, and also is the responsible of the NER (Named Entity Recognition). Understanding NER: You can have several entities defined, each one with multilanguage and several texts for each option. Example Entity Option English Spanish FOOD Burguer Burguer, Hamburguer Hamburguesa FOOD Salad Salad Ensalada FOOD Pizza Pizza Pizza
