Class: NerManager

NerManager(settings)

Class for a named entities manager, that can be recognized inside a string. Basically, a named entity is an enumerator that has a set of options, and each option can have several words to refer to this option. Example: We have the entity "Superhero". Inside "Superhero" we have different options: - Superman: Superman - Spiderman: Spiderman, Spider-man - Wolverine: Wolverine, Logan, Patch, Weapon X
